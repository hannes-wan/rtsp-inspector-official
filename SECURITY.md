# Security policy

## Report privately

Do not open a public issue for a suspected vulnerability in RTSP Inspector. Email [support@hannes-software.com](mailto:support@hannes-software.com) with the product version, operating system, impact, and the smallest safe reproduction you can provide.

Do not attach credentials, license files, patient records, private audio, packet captures, camera URLs, device identifiers, or other customer data to a public GitHub issue. We will reply with a private handoff path when files are necessary.

## Supported release

Security fixes target the current release listed on the [official download page](https://hannes-software.com/rtsp-inspector/download/). Reproduce against the current version when it is safe to do so.
