# RTSP Inspector support

Start with the [product help center](https://hannes-software.com/rtsp-inspector/help/).

Use GitHub Issues for reproducible product bugs and focused feature requests. Include the application version, operating system, expected behavior, actual behavior, and a minimal reproduction.

For licensing, billing, security, or cases containing private data, email [support@hannes-software.com](mailto:support@hannes-software.com). Never post license keys, credentials, patient information, private recordings, packet captures, or customer files in a public issue.
