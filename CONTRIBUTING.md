# Contributing to the RTSP Inspector community repository

Thank you for helping make RTSP Inspector more reliable and easier to use. The Community Edition is free, and this public repository is the right place for reproducible bug reports, focused workflow requests, and documentation corrections.

## What belongs here

- Reproducible product behavior with the app version, operating system, expected result, actual result, and the smallest safe set of steps.
- Focused feature requests that explain the real workflow problem and desired outcome.
- Documentation corrections, missing troubleshooting steps, accessibility problems, and broken public links.

## What does not belong here

- Credentials, license keys, patient records, private recordings, packet captures, camera URLs, device identifiers, or customer data.
- Billing, licensing, or security reports; send those privately to [support@hannes-software.com](mailto:support@hannes-software.com).
- Source-code pull requests. Product development happens in a private workspace, so public contributions are handled as issues and documentation feedback rather than implying that this distribution repository is open source.

Please use the structured issue templates. Maintainers may close duplicates, requests outside the product boundary, or reports that cannot be reproduced after a reasonable request for evidence.
